# Mavi Digital IA

Landing page en Next.js para vender el Pack Emprendedor IA.

## Cómo usarlo

1. Abrí la carpeta en Visual Studio Code.
2. Abrí la terminal dentro del proyecto.
3. Instalá dependencias:

```bash
npm install
```

4. Ejecutá el proyecto:

```bash
npm run dev
```

5. Entrá en el navegador:

```bash
http://localhost:3000
```
